Branchless Bank Case Study

Summary

This is a mobile application designed for a hypothetical online only bank. It is a streamlined design for an "on the go" user. Color scheme was chosen to simultaneously envoke a sense of security and welcomeness. A banking application is not usually the most enjoyable visit, so I decided to make a banking app that the user would want to come back to.

Challenges and Solutions

The biggest challenge I faced with this project was to design a simple, intuitive, and spatial interface. Finding space on a mobile app is not always easy, but I found that keeping only what was absolutely needed on the page at any one time helped keep it easy to use while maintaining open space. Visual heirarchy was key. Main command buttons were designed large so there would be no ambiguity as to how to complete a given task.

A second challenge I had was to convey a sense of security without sacrificing a sense of invitation. Color scheme and the right logo all played in to this. The gray is serious and stark, while the green logo is contrastingly organic and welcoming.
